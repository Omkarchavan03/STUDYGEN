# StudyGen - Educational Learning Platform 🎓

**StudyGen** is an educational platform designed for distraction-free learning. It provides video lectures, e-content (PDFs, images, and other study materials), courses (free and paid), and AI-powered study assistance to help students learn efficiently.

StudyGen combines the ease of video platforms with education-specific tools to create a user-friendly and engaging learning environment.

---

## 🚀 Features

### 🎥 Video Lectures
- Distraction-free playback
- Follow your favorite educators
- Smooth UI/UX optimized for learning

### 📚 E-Content Library
- Upload, access, and share PDFs, images, and other study materials
- Organized for easy browsing
- Supports all common study file types

### 🏫 Courses
- Free and paid courses
- Structured learning paths
- Access to course materials and video lessons

### 🤖 AI Study Assistance
- GenAI-powered help for learning and problem-solving
- Summarizes content, generates study notes, and answers questions
- Guides students through complex concepts

---

## 💻 Tech Stack

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: PHP, MySQL  
- **Server**: XAMPP (for local development)  
- **AI Integration**: API keys for GenAI study assistance  
- **Design**: Clean, intuitive UI with smooth UX for students and educators

---

## ⚙️ Installation

1. Clone the repository:
```
git clone https://github.com/omkarchavan03/studygen.git
```
1. Install **XAMPP** or another local server  
2. Import the MySQL database `studygen.sql`  
3. Configure `db.php` with your database credentials and AI API keys  
4. Open the platform in your browser via your local server
