# StudyGen – Education Platform for the New Generation 🎓

**StudyGen** is an online education platform designed for modern learners.  
It allows students to discover educational videos, access study materials, and learn from teachers through structured content.

The platform also allows educators to **monetize their knowledge** by uploading courses and educational videos, similar to a learning-focused version of YouTube.

StudyGen focuses only on **educational content**, making it easier for students to find reliable learning resources without distractions.

---

## 🚀 Features

### 📚 Educational Content
- Access a wide range of **educational videos**
- Study **e-content in PDF format**
- Download videos and notes for offline learning
- Search educational topics easily

---

### 👨‍🏫 Teacher & Creator System
- Teachers can **create their own channels**
- Upload educational videos and courses
- Sell paid courses to students
- Monetize content similar to a creator platform

---

### 👨‍🎓 Student Learning Experience
- User-friendly learning interface
- Follow favorite teachers
- Discover both **free and paid courses**
- Search topics like a video platform but focused only on **education**

---

### 🔐 Authentication System
- User registration and login system
- Personalized user accounts
- Track favorite teachers and learning content

---

### 🤖 StudyGen AI (GenAI Support)

StudyGen includes an **AI-powered learning assistant** that helps students understand content better.

AI Features:

- Generate **summaries of educational videos**
- **Question & Answer support**
- Explain complex topics in a simple way
- Assist students in quick revision

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/your-username/studygen.git
```

2. Move the project to your local server directory (XAMPP/WAMP/LAMP).

3. Import the **MySQL database**.

4. Start the local server.

5. Open the project in your browser.
